欢迎使用 Word MD Editor

这是一款模仿 Microsoft Word 界面的所见即所得 Markdown 编辑器。

它结合了 Word 的熟悉界面和 Markdown 的简洁语法，提供直观的编辑体验。

功能特点

  • 所见即所得 - 实时预览 Markdown 效果
  • Word风格 - 熟悉的 Ribbon 工具栏和界面
  • 完整格式 - 支持加粗、斜体、下划线、删除线、上标、下标
  • 段落格式 - 支持项目符号、编号列表、引用块、对齐方式
  • 样式设置 - 字体、字号、行高、字间距、颜色
  • 文件管理 - 新建、打开、保存、导出文件
  • 撤销重做 - 完整的历史记录
  • 查找替换 - 强大的查找和替换功能
  • 主题切换 - 浅色和深色主题
  • 自动保存 - 每30秒自动保存内容

开始使用

直接在编辑器中输入内容，使用上方工具栏格式化文本，或切换到 Markdown源码 视图直接编写 Markdown 语法。

提示： 尝试使用快捷键提高编辑效率！

格式快捷键

  • Ctrl + B - 加粗
  • Ctrl + I - 斜体
  • Ctrl + U - 下划线
  • Ctrl + 1-6 - 标题1-6级别
  • Ctrl + Z - 撤销
  • Ctrl + Y - 重做
  • Ctrl + F - 查找
  • Ctrl + S - 保存
  • Ctrl + O - 打开文件
  • Ctrl + N - 新建文件

插入元素

  • 表格 - 点击"插入" → "表格"，选择行列数
  • 链接 - 点击"插入" → "链接"，输入URL和显示文字
  • 图片 - 点击"插入" → "图片"，支持URL或本地上传
  • 水平线 - 点击"插入" → "水平线"

段落格式

  • 项目符号列表 - 按回车输入"-"开始列表
  • 编号列表 - 按回车输入"1."开始列表
  • 任务列表 - 点击"项目符号"后选择"任务列表"
  • 引用块 - 点击"引用"按钮
  • 对齐方式 - 左对齐、居中、右对齐、两端对齐

视图切换

  • 所见即所得 - 实时预览 Markdown 渲染效果
  • Markdown源码 - 查看和编辑原始 Markdown 代码
  • 标尺显示 - 控制左侧标尺的显示
  • 样式面板 - 控制右侧样式面板的显示
