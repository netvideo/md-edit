# Word MD Editor - 使用说明

## 项目概述
Word MD Editor 是一款模仿 Microsoft Word 界面风格的所见即所得 Markdown 编辑器。它结合了 Word 的熟悉界面和 Markdown 的简洁语法，提供直观的编辑体验。

## 功能特性

### 1. Word风格界面
- **Ribbon工具栏**: 模仿 Word 的选项卡式工具栏
- **标尺**: 左侧显示标尺，可控制缩进
- **样式面板**: 右侧显示当前格式信息
- **状态栏**: 底部显示文件名、字数、光标位置和视图模式

### 2. 格式功能
- **字体格式**: 加粗、斜体、下划线、删除线、上标、下标、高亮标记
- **段落格式**: 项目符号列表、编号列表、任务列表、引用块、对齐方式
- **标题样式**: 标题1-6级别，每个级别不同样式和间距
- **代码块**: 行内代码和多行代码块，深色主题
- **行高和字间距**: 多种行高和字间距选项

### 3. 文件操作
- **新建文件**: 创建空白文档
- **打开文件**: 打开.md、.html或.txt文件
- **保存文件**: 保存到本地
- **另存为**: 导出为HTML或Markdown文件

### 4. 插入功能
- **表格**: 自定义行列数插入精美表格
- **链接**: 插入超链接，带悬停效果
- **图片**: 支持URL链接或本地上传，带圆角和阴影
- **水平线**: 插入分割线

### 5. 视图模式
- **所见即所得**: 实时预览精美 Markdown 渲染效果
- **Markdown源码**: 查看和编辑原始 Markdown 代码
- **标尺显示**: 可隐藏/显示左侧标尺
- **样式面板**: 可隐藏/显示右侧样式面板
- **主题切换**: 浅色/深色主题

## 快捷键

### 格式快捷键
- `Ctrl + B`: 加粗
- `Ctrl + I`: 斜体  
- `Ctrl + U`: 下划线
- `Ctrl + 1-6`: 标题1-6级别
- `Ctrl + Z`: 撤销
- `Ctrl + Y`: 重做
- `Ctrl + F`: 查找
- `Ctrl + P`: 打印

### 文件操作
- `Ctrl + N`: 新建文件
- `Ctrl + O`: 打开文件
- `Ctrl + S`: 保存文件
- `Ctrl + Shift + Y`: 重做

## 使用方法

### 1. 文件操作

#### 新建文件
1. 点击"文件"选项卡
2. 点击"新建"按钮
3. 或使用快捷键：`Ctrl + N`

#### 打开文件
1. 点击"文件"选项卡
2. 点击"打开"按钮
3. 选择要打开的文件（支持.md、.html、.txt格式）
4. 或使用快捷键：`Ctrl + O`

#### 保存文件
1. 点击"文件"选项卡
2. 点击"保存"按钮
3. 或使用快捷键：`Ctrl + S`

#### 另存为
1. 点击"文件"选项卡
2. 点击"另存为"按钮
3. 选择保存位置和文件名
4. 支持导出为HTML或Markdown格式

### 2. 基本编辑
1. 直接在编辑区域输入内容
2. 选中文本后点击工具栏按钮应用格式
3. 使用样式下拉框切换标题级别

### 3. 插入元素
1. 点击"插入"选项卡
2. 选择要插入的元素类型
3. 在弹出的对话框中配置参数
4. 点击"插入"按钮

### 4. 视图切换
1. 点击"视图"选项卡
2. 勾选/取消勾选相应选项
3. 可同时显示/隐藏标尺和样式面板
4. 可切换所见即所得和Markdown源码视图

### 5. 查找替换
1. 点击状态栏的"查找"按钮或使用快捷键`Ctrl + F`
2. 输入查找内容
3. 点击"查找下一个"、"替换"或"全部替换"

## Markdown语法支持

| Markdown语法 | 对应效果 |
|-------------|----------|
| # 标题 | 一级标题 (28pt, 深色, 底部边框) |
| ## 标题 | 二级标题 (22pt, 中深色) |
| ### 标题 | 三级标题 (18pt, 中等深色) |
| **加粗** | 加粗文本 (Times New Roman 粗体) |
| *斜体* | 斜体文本 |
| ~~删除线~~ | 删除线文本 |
| `代码` | 行内代码 (浅灰背景) |
| ```代码块``` | 多行代码块 (深色主题, 圆角, 阴影) |
| - 列表项 | 无序列表 (圆点标记) |
| 1. 列表项 | 有序列表 (数字标记) |
| - [ ] 任务 | 任务列表 (空/勾选圆圈) |
| > 引用 | 引用块 (蓝色边框, 渐变背景) |
| [链接](url) | 超链接 (蓝色下划线) |
| ![图片](url) | 图片 (圆角, 阴影, 响应式) |
| --- | 水平线 (分割线) |
| ==高亮== | 高亮标记 (黄色背景) |
| | 表格 | 精美表格 (渐变表头, 斑马线, 悬停) |

## 渲染优化

### 所见即所得效果
编辑器提供精美的实时渲染效果：

1. **标题层次**: 每级标题都有独特的样式和间距
2. **段落布局**: 两端对齐，适当的行高
3. **列表样式**: 优美的标记和间距
4. **表格设计**: 圆角、阴影、悬停效果
5. **代码美化**: 深色主题，适合编程
6. **引用风格**: 渐变背景，蓝色边框
7. **图片美化**: 圆角、阴影、响应式缩放
8. **链接样式**: 蓝色下划线，悬停效果

### 主题切换
- **浅色主题**: 默认清爽风格
- **深色主题**: 保护视力的夜间风格

### 交互效果
- 表格行悬停高亮
- 图片悬停放大
- 链接悬停颜色变化
- 流畅的过渡动画

## 技术说明

### 文件结构
```
word-md-editor/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── editor.js       # 编辑器逻辑
└── SPEC.md         # 规格说明书
```

### 浏览器兼容性
- 支持现代浏览器（Chrome 80+, Firefox 75+, Edge 80+）
- 需要支持 contenteditable 属性
- 需要支持 FileReader API（用于图片上传）

### 本地运行
1. 下载所有文件到同一目录
2. 用浏览器打开 index.html
3. 无需服务器环境

## 注意事项

1. **图片上传**: 仅支持本地预览，不会上传到服务器
2. **内容保存**: 文件保存到本地，不会自动保存到云端
3. **自动保存**: 每30秒自动保存到浏览器localStorage
4. **浏览器限制**: 某些高级格式功能可能受浏览器限制
5. **性能**: 大文档编辑时建议切换到Markdown源码视图
6. **文件格式**: 支持.md（Markdown）、.html（HTML）、.txt（文本）三种格式
