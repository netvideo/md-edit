# Word风格Markdown编辑器 - 规格说明书

## 1. 项目概述

- **项目名称**: Word MD Editor
- **项目类型**: Web富文本编辑器
- **核心功能**: 模仿Microsoft Word UI风格的所见即所得Markdown编辑器，支持实时Markdown转换和丰富的排版功能
- **目标用户**: 需要Word般编辑体验的Markdown用户

## 2. UI/UX 规格

### 2.1 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  顶部功能区 (Ribbon)                                          │
├──────┬──────────────────────────────────────────┬───────────┤
│      │                                          │           │
│ 标尺 │           编辑/预览区域                    │  样式面板  │
│      │                                          │           │
│ 24px │              600px                       │   200px   │
│      │                                          │           │
├──────┴──────────────────────────────────────────┴───────────┤
│  底部状态栏                                                  │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 色彩方案

- **主色**: #2B579A (Word蓝)
- **强调色**: #106EBE
- **背景色**: #FFFFFF
- **工具栏背景**: #F3F2F1
- **边框色**: #E1DFDD
- **悬停色**: #C7E0F4
- **文字色**: #323130
- **次要文字**: #605E5C

### 2.3 字体

- **编辑器字体**: "Calibri", "Microsoft YaHei", sans-serif
- **代码字体**: "Consolas", "Courier New", monospace
- **字号范围**: 11px - 36px

### 2.4 顶部功能区 (Ribbon)

分为以下几个选项卡：

#### 开始 (Home)
- 字体组：字体名称、字号、加粗、斜体、下划线、删除线、下标、上标
- 段落组：项目符号、编号、等级、缩进、水平线
- 样式组：正文、标题1-9、代码块

#### 插入 (Insert)
- 表格组：插入表格
- 链接组：插入链接
- 图片组：插入图片
- 分割线

#### 视图 (View)
- 显示/隐藏标尺
- 显示/隐藏样式面板

### 2.5 侧边样式面板

显示当前光标所在位置的格式信息：
- 字体名称和字号
- 当前样式（标题级别）
- 快捷操作按钮

### 2.6 底部状态栏

- 字数统计
- 光标位置（行、列）
- 当前视图模式

## 3. 功能规格

### 3.1 格式功能

| 功能 | 快捷键 | 按钮位置 |
|------|--------|----------|
| 加粗 | Ctrl+B | 字体组 |
| 斜体 | Ctrl+I | 字体组 |
| 下划线 | Ctrl+U | 字体组 |
| 删除线 | - | 字体组 |
| 下标 | - | 字体组 |
| 上标 | - | 字体组 |
| 标题1-9 | Ctrl+1~9 | 样式组 |
| 代码 | - | 样式组 |

### 3.2 段落功能

| 功能 | 快捷键 | 按钮位置 |
|------|--------|----------|
| 无序列表 | - | 段落组 |
| 有序列表 | - | 段落组 |
| 减少缩进 | - | 段落组 |
| 增加缩进 | - | 段落组 |
| 水平线 | - | 段落组 |

### 3.3 插入功能

- 插入表格（支持选择行列数）
- 插入链接（URL + 显示文字）
- 插入图片（URL或本地）
- 插入水平线

### 3.4 Markdown对应关系

| Markdown | 渲染效果 |
|----------|----------|
| # H1 | 标题1 (28pt, 粗体) |
| ## H2 | 标题2 (22pt, 粗体) |
| ### H3 | 标题3 ( 18pt, 粗体) |
| **** 加粗 ** | 加粗文本 |
| *斜体* | 斜体文本 |
| ~~删除线~~ | 删除线文本 |
| `代码` | 等宽字体+浅灰背景 |
| ```代码块``` | 多行代码块 |
| - 无序列表 | 项目符号列表 |
| 1. 有序列表 | 数字编号列表 |
| > 引用 | 左侧边框+灰色背景 |
| --- | 水平线 |
| 链接 | 蓝色下划线 |
| 图片 | 图片渲染 |

## 4. 交互规格

### 4.1 编辑器行为

- **内容可编辑**: 使用 contenteditable 实现
- **光标操作**: 支持光标移动、选中文本
- **格式应用**: 选中文本后点击按钮应用格式
- **Markdown源**: 底部切换查看Markdown源码

### 4.2 工具栏交互

- 按钮悬停时高亮
- 当前激活的格式按钮保持高亮状态
- 下拉框选择字体和字号
- 弹出式对话框插入链接/表格/图片

### 4.3 响应式行为

- 最小宽度: 800px
- 编辑区域自适应
- 侧边面板可收起

## 5. 技术实现

- 纯HTML/CSS/JavaScript实现
- 使用 contenteditable 作为编辑器核心
- Markdown解析使用正则表达式实时转换
- 支持本地图片预览（FileReader API）
